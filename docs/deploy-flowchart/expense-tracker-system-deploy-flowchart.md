# Deploy Flowchart

เอกสารนี้ใช้สำหรับอธิบายลำดับขั้นตอนการ Deploy แบบกึ่งอัตโนมัติของระบบ โดยอ้างอิงจากสคริปต์ `deploy.sh` ที่ใช้งานจริงในโครงงาน

## ภาพผังกระบวนการ Deploy

1. [Deploy Flowchart (PNG)](./Deploy%20Flowchart.drawio.png)

## คำอธิบายโดยสรุป

1. เริ่มกระบวนการด้วยการรันสคริปต์ deploy.sh
2. แจ้งสถานะเริ่มต้นผ่าน Discord Webhook
3. หยุดบริการเดิม ดึงโค้ดล่าสุด สร้างอิมเมจใหม่ และเริ่มบริการอีกครั้ง
4. ทำความสะอาด Image/Cache ที่ไม่ใช้งานเพื่อลดการใช้พื้นที่
5. แจ้งผลสำเร็จหรือความล้มเหลวผ่าน Discord ตามผลลัพธ์ของกระบวนการ
